# preparacion_certamen_4

## 1.- Creamos el proycto en git
- Abrimos Git, creamos un nuevo repositorio
- Abrimos la carpeta donde creamos el repositorio
    y ejecjutamos el VisualStudio.


## 2.- Creamos el proyecto para Vue.js
    - El "." significa aquí mismo, darle yes/y cuando pregunte
    - npm create vite@latest . -- --template vue
    - ignore files and continue
    - install with npm and start now

## 3.- Instalar dependencias:
    - npm install

## 4.- Ejecutamos el proyecto
    - npm run dev

## 5.- Removemos los archivos inecesarios, los cuales son:
